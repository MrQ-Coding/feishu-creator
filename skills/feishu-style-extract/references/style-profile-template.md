confidence: medium

## Review Checklist

- 样本是否至少覆盖 `3` 篇原创文档
- 结论是否把主题词和风格词区分开了
- `风格指纹` 是否足够具体，能直接指导后续写作
- 是否明确标出了低置信度判断
- 如果要保存回飞书，是否已经先给用户看过草稿

## Save-Back Suggestions

- 新建页面时，标题可用 `✍️ 写作风格画像`
- 如果页面已经存在，优先更新固定章节而不是重复创建新页面
- 保存前先确认用户是否要保留 `样本范围` 和 `代表性片段`
- 共享 HTTP 场景下，尽量写入 `owner_id` 和 `profile_kind`，不要只靠标题区分画像
