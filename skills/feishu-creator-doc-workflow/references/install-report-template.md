# Install Report Template

Use this shape when summarizing setup results for the user:

- `安装结果`
- `环境`
- `仓库位置`
- `执行结果`
- `生成或更新的文件`
- `启动冒烟测试`
- `仍需手动填写`
- `下一步建议`

Keep file paths absolute, mention whether proxy variables were detected, and explicitly list missing Feishu credentials or renderer prerequisites when setup is not fully complete.
